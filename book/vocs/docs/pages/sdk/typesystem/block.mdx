# Block Types

Understanding Reth's block type system and representations.

## Overview

Reth provides several block types for different use cases and contexts within the Ethereum execution layer.

## Block Representations

### Sealed Block
Blocks that have been validated and sealed.

### Block Body
The transaction list and other block contents.

### Block Header
Block metadata including parent hash, state root, etc.
