# Reth as a Library

This section covers how to use Reth as a library in your own applications.

## Introduction

Reth is designed to be modular and composable, allowing developers to use its components as building blocks for custom Ethereum applications and nodes.

## Key Benefits

- **Modular Architecture**: Use only the components you need
- **Performance**: Built with performance in mind using Rust
- **Flexibility**: Customize behavior to fit your specific use case
- **Standards Compliant**: Implements Ethereum specifications correctly

This section will guide you through:

- Understanding Reth's type system
- Learning about node components
- Building custom nodes
- Working with standalone components
